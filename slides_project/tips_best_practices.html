Dicas de Ouro para uma configuração sem erros

Limpeza de Registros

Remova registros do tipo A ou CNAME antigos que apontem para outros servidores para evitar conflitos de rota.

Ajuste de TTL

Mantenha o TTL (Time To Live) em "Automático" ou "3600 segundos". Isso equilibra a velocidade de atualização com a estabilidade do site.

Limpeza de Cache

Se o site não abrir após a propagação, limpe o cache do navegador ou use uma aba anônima para forçar a busca dos novos dados de DNS.

Dados Atualizados

Mantenha seu e-mail de cadastro no registrador sempre atualizado para não perder avisos críticos de renovação de domínio.
