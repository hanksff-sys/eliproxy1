Passo 2: Obtendo o DNS Target da Hospedagem

1. Adicione o Domínio

No painel da sua hospedagem (ex: Heroku Settings), localize a seção de domínios e adicione www.eliproxy.com.br.

2. Localize o Endereço

Após adicionar, o sistema gerará automaticamente um endereço de destino específico para o seu site.

3. Copie o Valor

Copie este endereço exatamente como ele aparece. Você o usará no próximo passo dentro do seu registrador de domínio.

Exemplo de DNS Target

eliproxy-7a2b9.herokudns.com

Dica: Este endereço é único para o seu projeto e funciona como o "ID" do seu servidor na rede global.
